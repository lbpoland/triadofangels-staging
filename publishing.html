How to add books (fast)

  1. Edit: /js/publishing-data.js — add/update sagas, series, and books.
  2. Add covers: place images under /assets/images/books/ (square + wide).
  3. Add samples/exports: place files under /assets/publishing/ (EPUB/PDF/etc) and reference paths in the book record.
  4. Done: the library grid + book page render automatically from the data.
